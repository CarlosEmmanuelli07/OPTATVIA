# Proyecto de Prueba

Este proyecto es un **ejemplo** para practicar Markdown y documentación de proyectos en Git.

---

## Descripción

El propósito de este proyecto es:

- Aprender a documentar un proyecto en Markdown.
- Practicar el uso de Git y GitHub.
- Mantener organizada la información del proyecto.
